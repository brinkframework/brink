# Brink Framework

## Installation

    $ pip install brink

## Getting started

Getting started is really easy assuming you already have RethinkDB up and running. You might need to review the `config.py` file, to ensure it's compatible with your setup.

    $ brink start-project myproject
    $ cd myproject
    $ brink sync-db
    $ brink run
